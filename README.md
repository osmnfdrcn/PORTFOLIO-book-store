# PORTFOLIO : BOOK STORE

## Environment Variables

To run this project, you will need to add the following environment variables to your .env file
`DATABASE_URL`

## Run

```bash
  git clone https://github.com/osmnfdrcn/PORTFOLIO-book-store.git
  npm install
  npm run start
```
